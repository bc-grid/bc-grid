# Licence — TBD

The licence model for bc-grid is undecided. Possibilities under consideration:

- **MIT / Apache-2.0** — open source, no commercial product
- **Dual-licence (MIT + commercial)** — TanStack model; free for most, paid for enterprise features
- **Source-available** — code visible, commercial licence required (Sentry / Bitwarden model)
- **Proprietary** — closed source, sold as a product (AG Grid model)

**Until decided**: treat the codebase as proprietary. Do NOT open-source any commits without explicit approval. All contributions assume eventual licence flexibility, so dependencies must be permissively licensed (MIT / Apache-2.0 / BSD / ISC) — no GPL, AGPL, or copyleft of any kind.

The licence decision is deferred until there's product-market signal (e.g., active beta cohort) that informs the business model.
