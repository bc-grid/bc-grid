# @bc-grid/enterprise

Enterprise-only bc-grid extensions. **v0.1.0-alpha.1 ships an empty placeholder** to lock the `@bc-grid` namespace.

Reserved for Q5 enterprise features (advanced server-row-model integrations, audit-grade export, role-based column visibility) that don't fit the OSS-style open layer of `@bc-grid/react`.
